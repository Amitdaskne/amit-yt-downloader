AMIT YT DOWNLOADER 🔥

Simple Termux-based YouTube & Facebook video downloader using yt-dlp.

🚀 Installation (Termux)

pkg update && pkg upgrade
pkg install python git yt-dlp -y
termux-setup-storage

📥 Clone Project

git clone https://github.com/YOUR_USERNAME/amit-yt-downloader.git
cd amit-yt-downloader

▶️ Run

python yt.py

📁 Output Folder

Videos will be saved in:

/storage/emulated/0/YT VIDEO

❌ Exit

Type "exit" to quit
